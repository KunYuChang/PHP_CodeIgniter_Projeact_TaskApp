# PHP_CodeIgniter_Projeact_TaskApp

## 建立專案

```php
composer create-project codeigniter4/appstarter task-app
```

## 設置開發模式

🤔 要幹嘛?

1. 可顯示 Debug 訊息
2. 可使用 Debug 工具

🤔 如何設定?

1. 專案剛建立時，會有一個`env`檔案，改名為`.env`。
2. 檔案內容更改為 : `CI_ENVIRONMENT = development`

- 這是一個環境變數設定檔。(Environment Configuration file)
- `.env`是框架才有的設計
